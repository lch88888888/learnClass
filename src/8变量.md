### 变量和常量
>变量：值是可以变得
>常量：值是不可以变得

#### 怎么定义一个变量
```
	 //js中定义变量的方式 就是用一个var定义
	 //var 变量 = 值;
	 var str = 'hello word'
	 var num = 4
	 console.log(num) // 4 变量只是一个无意义的名字，它所代表的意义都是其存储的那个值
	 num= 3;//让原有的num变量储存的值修改为13（变量代表一个值）
	 
```
